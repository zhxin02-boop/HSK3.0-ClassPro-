# HSK1 课程内容与题库命题原则 v1

## 一、课程内容原则

- 每课围绕教材课题、生词、语法和课文生成，不脱离本课教学目标。
- 课文原句优先保留；新增例句、练习句和情境对话必须标记为“教学扩展”。
- 先保证核心闭环：课前预习 → 课中操练 → 课后巩固 → 教师反馈。
- 题目要服务于词汇识别、语法理解、交际使用和综合输出，不只做词义记忆。

## 二、跨课复现原则

从第二课开始，每课内容按以下比例复现已学内容：

- 约 60%：本课新词、新语法和新课文。
- 约 25%：上一课内容，保证及时复习。
- 约 15%：前面多课的累计复习，逐渐形成长期练习池。

复现内容不应只是重复原题，而应更换语境、题型或任务。例如：

- 课前：快速识别和辨义。
- 课中：在对话、竞赛和句型操练中调用旧知识。
- 课后：把新旧词汇放在同一交际任务中输出。

## 三、题库结构原则

每道题必须具备：

- 唯一题目 ID
- 课程归属
- 来源类型：本课新知 / 上课复习 / 累计复习
- 环节：课前 / 课中 / 课后
- 题型、难度、知识点
- 客观题答案或主观题批改要求

题库分为三层：

1. 课程核心题：紧贴本课教材，保证每课闭环。
2. 累计复习题：跨课复现旧词、旧语法和旧交际功能。
3. 拓展题：用于 B/C 档作业、课堂挑战和教师选择性使用。

## 四、难度与分层

- A 档：识别、选择、简单填空，保证基础学生完成。
- B 档：句型操练、短对话和受控输出，作为标准作业。
- C 档：综合情境、自由表达和迁移任务，需要教师批改。

## 五、审核标记

内容草案中使用以下标记：

- `教材原文`：来自教材或用户提供的原始数据。
- `教学扩展`：根据教学目标生成，需教师审核。
- `待确认`：存在人物、翻译、答案或语境歧义。

未经审核的扩展内容不直接替换已上线课程。
