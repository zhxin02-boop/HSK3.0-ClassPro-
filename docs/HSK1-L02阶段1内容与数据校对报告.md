### 2. 语法例句修正

位置：

- `source/in-class/data_L02.js`

当前文本：

```text
我是学生。不我是学生。
他叫李文。不他叫李文。
认识你很高兴。认识你也高兴。
```

问题：

- `不我是学生`、`不他叫李文` 是错误示范形式，但当前页面没有明确标注“错误形式”，学生可能误学。
- `认识你也高兴` 不如课文中的 `认识你我也很高兴` 自然。

建议改为：

```text
我是学生。 -> 我不是学生。
他叫李文。 -> 他不叫李文。
很高兴认识你。 -> 认识你我也很高兴。
```

需要教师确认：

- 已确认：不要直接展示错误句，改为正确对比形式。

已执行修正：

```text
我是学生。 -> 我不是学生。
他叫李文。 -> 他不叫李文。
很高兴认识你。 -> 认识你我也很高兴。
```

### 3. 连词成句题 g1_hsk1_02_03 修正

位置：

- `source/in-class/data_L02.js`

当前题目：

```js
chunks: ["我","不是","安妮","是"]
correct_answer: "我不是安妮，我是白家月。"
```

问题：

- 词块里没有 `白家月`，学生无法拼出参考答案。
- 多出的 `是` 也不能自然组成完整句。

建议改为二选一：

方案 A，降低难度：

```js
chunks: ["我","不是","安妮"]
correct_answer: "我不是安妮。"
```

方案 B，完整还原课文：

```js
chunks: ["我","不是","安妮","我","是","白家月"]
correct_answer: "我不是安妮，我是白家月。"
```

需要教师确认：

- 已确认：第一轮低压力处理，采用方案 A。
- 已补充：`安妮` 在提示中增加拼音支架 `Ānnī`。

已执行修正：

```js
prompt_cn: "把下面的词排成正确的句子（安妮 Ānnī）"
chunks: ["我","不是","安妮"]
correct_answer: "我不是安妮。"
```

## 四、字段与数据结构问题

### 1. 课程标识不完全统一

当前存在：

- `Lesson 2`
- `L02`
- `HSK1-L2`
- `HSK1-L02`
- `hsk1_02`

建议：

- 对外提交、Sheets、GAS、教师批改统一使用 `HSK1-L02`。
- 课中内部显示可保留 `L02`。
- 后续新增字段 `lessonKey: "HSK1-L02"`，避免页面显示名和数据存储名混用。

### 2. 词汇字段不统一

课前 JSON：

```json
word, pinyin, english
```

课中 JS：

```js
hanzi, pinyin, english, pos, audio, tags
```

建议：

- 短期不强行合并。
- 中期以课中 `vocabulary` 的丰富结构为主，课前页面通过适配器读取。
- 后续统一字段使用 `hanzi`，兼容 `word`。

### 3. 题型数据字段基本可用

`data_L02.js` 的题目大多包含：

- `id`
- `type`
- `stage`
- `difficulty`
- `time_estimate_sec`
- `prompt_cn`
- `prompt_en`
- `data`
- `correct_answer`
- `kp`

建议补充：

- `lesson: "HSK1-L02"`
- `module`
- `reviewStatus` 仅在提交记录中使用，不必放入题库。

### 4. TSV 使用规范

当前数据源分工：

| 文件 | 状态 | 用途 |
|---|---|---|
| `source/in-class/data_L02.js` | 权威主数据源 | 页面运行、题型接入、后续模块化 |
| `source/data-model/HSK1-L02_题库数据_current.tsv` | 当前导出版 | Google Sheets 导入、人工审核 |
| `source/data-model/HSK1-L02_题库数据.tsv` | 废弃旧版 | 仅作历史参考，不再导入 |

原则：

- 后续先改 `data_L02.js`。
- 需要导入 Sheets 时，再从 `data_L02.js` 重新导出 TSV。
- 不再手工维护多个互相冲突的题库 TSV。

### 3. 本周建议

第一轮闭环保持：

- 必跑：词汇、语法、课文、选词填空、连词成句、PK。
- 优先接入：图片猜词、完成句子、课文判断，三者最适合快速接入。
- 第二批接入：词义/句子匹配、翻牌配对、阅读选择、课文填空复述。
- 第三批接入：听力系列、汉字部件、拼音到汉字。它们适合 HSK1，但依赖音频、交互或后续拼音学练模块。
- 原则：这些都是 HSK1 课中题型，不删除；只按实现难度安排进入样课的先后顺序。
