# ClassPro HSK1 课程内容与练习命题原则 v2

## 一、内容总原则

1. 课文原句必须保留。课文、对话、人物和教材给出的例句不得为了“题目变化”而删除或改写。
2. 新增例句、练习句、情景对话和扩展文本必须标记为“教学扩展”，不得与教材原文混淆。
3. 练习服务于“能理解、可辨认、会运用、愿交流”，不考察学生只识记语言知识。
4. 练习范围包括本课新知和已学内容。后续课程必须及时复现已学词汇、已学语法和已学交际功能。
5. HSK1 题目优先使用学生已经学过的词汇；确需出现新词时，应提供中文、图片或英文辅助，不让非目标词汇成为主要障碍。
6. 每道题必须能说明“为什么要出”。如果不能对应词汇、语法、课文理解、交际功能或跨课复习目标，则不进入正式题库。
7. 选择题选项的拼音统一显示在汉字正上方，禁止把拼音放在汉字右侧或混用两种位置。除汉字辨析题外，汉字选项需要提供拼音；汉字辨析题的选项统一只显示汉字，不显示拼音。
8. 同一道题的选项格式必须完全一致，不得出现重复选项、同一答案的不同显示格式或拼音缺失。

## 二、跨课复现原则

从 L02 开始，默认每课练习构成为：

- 约 60%：本课新词、新语法和本课课文。
- 约 25%：上一课内容，保证及时复习。
- 约 15%：更早课程的累计复习。

比例可以按课次调整，但每课的课前、课中、课后都必须出现已学内容：

- 课前：快速识别已学词汇、已学句型和已学交际表达。
- 课中：把已学内容放进新的对话、配对、竞赛和句型操练。
- 课后：在已学内容与本课新内容共同组成的生活任务中输出。

复现不等于机械重复。相同知识点应尽量改变题型、人物、场景或任务；但课文原句本身必须保留，并在适合的题型中承担复习作用。课前、课中、课后都要把已学内容放进新的练习中，不能只在某几个题型中复现。

## 三、每道题的必备命题信息

每道题除运行字段外，还必须保存以下审核字段：

```json
{
  "sourceType": "教材原文|教学扩展|跨课复习",
  "targetKnowledge": ["本题要测的词汇或语法"],
  "principle": "为什么设计这道题，以及它如何符合本题型原则",
  "difficultyReason": "为什么属于 A/B/C 或简单/中等/挑战",
  "contentStatus": "待审核|已审核"
}
```

课文原句还应增加：

```json
{
  "isTextbookOriginal": true,
  "textSource": "用户提供的教材原文"
}
```

## 四、HSK1 题型命题原则

### 1. 选择题 choice

- 用于课前快速识别、课中即时检查和课后基础巩固。
- 题干短、目标单一，避免同时考多个新词。
- 干扰项必须来自同课或已学内容，且有可解释的错误原因。
- 不用生僻英文或复杂中文制造难度。
- 课前以识别为主，课中逐步加入语境，课后加入简单交际判断。

### 2. 选词填空 fill_blank / vocab_fill

- 句子必须是自然语境，不为填入目标词而制造生硬句子。
- 每道题主要测一个词或一个语法点。
- 备选词之间应有词义、词性或搭配上的可辨析性。
- 至少一部分题使用教学扩展句，不全部照搬课文例句。

### 3. 连词成句 ordering

- 每个核心语法点建议 6—10 题，按难度递进。
- 必须保留一部分课文原句或教材原句作为复现题，并标记 `教材原文`。
- 其他题必须是教学扩展句，使用本课语法和已学词汇。
- 前几题短句，后几题逐步增加词块数量；词块切分不能过度明显。
- HSK1 优先考基本语序、疑问句、否定句、数量和简单交际句型。

### 4. 完成句子 complete_sentence

- 留白必须让学生进行有意义的组织，而不是只填一个生词。
- 但 HSK1 仍需提供足够语境，避免开放程度过高导致无法判断。
- 题目应标明参考答案和可接受答案，必要时交教师批改。
- 参考答案只供教师使用，不直接展示给学生。

### 5. 看图猜词 image_guess

- 图片必须清楚、单一、可识别，适合课堂快速反应。
- 优先采用真实的课堂、校园和日常交际场景图片；插画或示意图只能作为暂时占位，不能作为最终上线素材。
- 每课覆盖核心新词，但不要求所有词都必须配图。
- 图片线索不能依赖复杂文化背景或学生尚未学过的词汇。

### 6. 描述猜词 / 你说我猜

- 描述必须使用学生已经学过或页面提供的辅助词。
- 描述角度要变化：功能、场景、动作、搭配或简单对比。
- 不能只是字典式翻译，也不能直接说出答案的一部分。
- 目标可以是单词、短语或简单句，不局限于单个生词。

### 7. 盲盒造句 / 输出题 output

- 词池应覆盖本课核心词，并分为 2 词、3 词、4 词等难度层级。
- 题目要给学生足够造句空间，不把答案限制成唯一模板。
- 开放输出必须标记教师批改；参考答案只是评分参考，不是唯一答案。
- B 档做受控输出，C 档做综合交际输出。

### 8. 挖空复述 / 课文理解

- 必须基于课文原文，优先挖核心实词、人物关系和目标语法。
- 避免只挖“的、了、在”等低价值虚词。
- 每题建议 1—3 个空，覆盖不同知识点。
- 可以增加理解性改写题，但必须保留对应的教材原句作为基础复现。

### 9. 课文判断与理解题

- 不能全部是原文逐句判断，要加入简单理解和信息对应。
- 错误选项要有最小但明确的变化，例如人物、对象、数量或动作变化。
- HSK1 不做复杂推理，保证学生能依据课文和图片完成判断。

### 10. 任务卡、接龙和综合活动

- 场景应来自学生真实生活：课堂、校园、宿舍、购物、问候、感谢、告别等。
- 基于课文场景，但不能要求学生原样背诵课文才能完成。
- 任务中要明确目标词、目标句型和允许使用的辅助词。
- starter 必须有延伸空间，不能一句话就结束活动。

## 五、HSK1 当前题型与题量建议

本版只针对当前 HSK1-L02 已经使用的题型，不提前引入 HSK3—HSK5 的题型体系。题量指“可用题库量”，不是一次课堂必须全部使用的数量。

### HSK1 当前 ClassPro 实施基准（2026-07-23）

为便于 L04 及后续课程直接复用统一数据契约，当前课程制作以以下规则为准；本节优先于上表中较早的递增建议：

- 看图猜词、选词填空、词句匹配、汉字区辨、句型转换、阅读/判断等主要固定题型：每课题库 10 题。
- 连词成句：每课 5 题。
- 词了个词：至少 3 组；按生词量拆分。20 个生词的课程建议 5 组 × 4 词。
- 课后作业：A、B、C 三档均提供 12 题；主观题必须显示可输入的答题框。
- 题库数量是可抽取题库量；页面可按课堂流程分批使用，但不得用重复题目凑数。

### 题量递增策略

- L01—L03：保持 L02 的基准数量，优先保证题型完整、跨课复现和内容质量。
- L04 及之后：继续保持固定题型 10 题、连词成句 5 题的正式课程基准；如需增加题量，必须先讨论并更新数据契约。
- 题库扩展优先增加不同语境和不同能力维度，不通过重复句子或重复选项凑数。
- 题库增加后，课堂页面应支持抽题或选择题组，不要求学生一次完成全部题目。
- 每个核心知识点至少准备“教材原句复现题 + 教学扩展题 + 跨课复习题”三类内容。

其他 HSK 级别的题型和数量另行制定，不在本 HSK1 版本中提前展开。

### 题量递增策略

- L01：建立基础题型覆盖，不追求题量最大化。
- L02—L03：补齐 L01/L02 的已学内容复现，并逐步增加同一知识点的不同语境题。
- L04 起：每课新增题量，同时扩大累计复习池；不把所有增长都放在本课新词上。
- 同一知识点至少准备“教材原句复现题 + 教学扩展题 + 跨课综合题”三类题目。
- 课堂实际呈现可以抽题，题库数量增加不等于课堂时长线性增加。

## 六、审核流程

1. 先审核教材原文是否完整、人物和翻译是否正确。
2. 再审核教学扩展是否符合 HSK1 词汇和语法范围。
3. 再审核答案、可接受答案和教师批改要求。
4. 最后在审核界面实际走一遍学生端和教师端。
5. 未审核内容不得覆盖已经可运行的 L02 闭环。
