# HSK1 样课闭环任务清单

> 本清单服务于本周目标：完成 HSK1 样课闭环，同时产出轻量版产品骨架确认稿。默认样课建议使用 `HSK1-L02`，因为现有材料和原型最多。

## 一、总目标

跑通一节 HSK1 样课的最小闭环：

`预习提交 -> Google Sheets 存储 -> 教师看板/批改 -> 课中互动 -> 课后反馈查看`

第一版不追求全功能，只证明这条链路真实可上课、可观察、可反馈、可复盘。
