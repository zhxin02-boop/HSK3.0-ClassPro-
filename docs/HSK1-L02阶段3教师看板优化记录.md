# HSK1-L02 阶段 3 教师看板优化记录

> 本记录对应《HSK1 样课闭环任务清单》的阶段 3：教师批改与课前看板。

## 一、真实测试结果

测试姓名：`Test_zhang`

已验证：

- 教师批改页能拉取 `HSK1-L02` 的提交记录。
- 客观题显示为 `无需批改`。
- 开放题显示为 `待批改`。
- 教师批改后，Google Sheets 状态能写回为 `已批改`。
- 学生端能查到教师批改记录。

## 二、发现的问题

教师批改页原本更像“数据表”，缺少课前快速判断信息。

实测中还发现：

- 得分框和反馈框提示不清楚。
- 教师容易把文字反馈写进得分处。
- 页面拉取的是所有课程数据，缺少 `HSK1-L02` 筛选。

## 三、已完成优化

文件：

- `source/teacher-dashboard/review.html`

新增：

- 页面标题改为 `教师批改与课前看板`。
- 默认筛选课程：`HSK1-L02`。
- 增加课前快速判断卡片：
  - 完成人数。
  - 提交记录数。
  - 客观题正确率。
  - 待批改数量。
- 增加高频薄弱点区域。
- 批改输入框增加提示：
  - 反馈框：`写给学生的反馈`。
  - 得分框：`0-10分`。
- 得分框改为数字输入。
- 批改成功后同步更新页面里的得分和状态。

## 四、当前教师端可回答的问题

打开教师批改页后，教师可以快速看到：

1. 有多少学生完成了。
2. 一共有多少条提交。
3. 客观题正确率大概是多少。
4. 还有多少开放题待批改。
5. 高频薄弱点集中在哪里。

这已经满足第一版课前看板的最低要求：教师能在课前快速判断今天要重点讲什么。
