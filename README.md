# Chinese ClassPro（HSK3.0） — 新HSK教学一体化生态系统

> 从学生真实数据出发，向闭环生态演进。

## 项目定位

**「课前—课中—课后」三位一体教学闭环**，以数据为纽带，以 AI 为引擎。基于 ClassReady + ClassPro 五年迭代经验的全新升级——不再是一个预习工具加一个课堂工具的简单拼盘，而是一体化的教学生态系统。

## 设计起点：学生访谈驱动

本项目的架构设计并非从零开始拍脑袋，而是建立在真实的学生反馈之上：

- **数据来源**：本学期收集的《访谈-平台升级建议》文档，包含了学生对 ClassReady / ClassPro 功能构想的直接反馈
- **核心发现**：学生明确提出了写作练习、发音纠正、AI 对话伙伴、汉字书写、离线可用、更多趣味游戏等高频需求
- **转化路径**：这些需求直接映射到三阶段闭环的数据模型中——课前暴露盲点、课中差异化互动、课后 AI 赋能巩固
- **关键决策**：不做"功能堆砌"，而是让数据贯穿始终，用数据驱动教学决策

> 升级建议现做参考，不是非要采用。至于如何升级，是我们结合现状需要做的实际决策。
