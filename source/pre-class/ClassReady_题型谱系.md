# ClassReady 题型谱系 v1.1

> 课前预习 · 轻量衔接 · 激趣为先 · 暴露盲点
>
> 与 ClassPro (HSK3.0) 课中题型谱系配套，构成完整闭环

---

## 设计总原则

### 核心理念

"**兴趣先行，盲点自现。**" ClassReady 不是考试，是让学生带着好奇心和方向感走进课堂。

### 五大原则

| 原则 | 说明 |
|---|---|
| ① 兴趣驱动 | 预习从好玩开始——话题预热、游戏化、文化感知，让学生想点进来 |
| ② 衔接性 | 每一道预习题都在为课中 ClassPro 的某个题型铺路 |
| ③ 轻量性 | 全课预习 ≤ 15 分钟，绝不给学生压力 |
| ④ 暴露性 | 不设分数压力，真实记录第一次尝试——这就是基线数据 |
| ⑤ 分级适配 | 不同 HSK 级别重点不同：HSK1 重拼音/汉字基础，HSK4/5 重篇章/写作 |

### 预习模块的完整流程

```
Step 1  话题预热  R0  ─── 文化感知 + 运用预知 → "这节课学完你能做什么？"
Step 2  生词闯关  R1  ─── 全部生词，游戏化匹配
Step 3  跟读练习  R2  ─── 读生词 + 语法句，系统评分
Step 4  [HSK1专]  拼音拼读 + 书写  R4/R5
Step 5  [HSK2专]  书写练习  R5
Step 6  课文听力  R6  ─── 听短句/篇章，答理解题
```

### 与 ClassPro 的衔接关系

| ClassReady（课前） | → | ClassPro（课中） |
|---|---|---|
| R0 话题预热 | → | 课上讨论/导入环节 |
| R1 词义闯关（全部生词） | → | v1 看图猜词 / v7 词了个词 |
| R2 跟读评分 | → | v2 你说我猜 / v6 描述猜词 |
| R4 拼音读写 | → | w2 看拼音写汉字 |
| R5 汉字书写 | → | w1 部件组字 |
| R6 课文听力 | → | t1 挖空复述 / tt 课文判断 / t3 接龙造句 |

## 一、题型总谱

---

### R0 话题预热 (topic_warmup)

| 属性 | 值 |
|---|---|
| 代码 | R0 |
| 主练能力 | 综合（听/说/文化感知）|
| 状态 | live |
| 时间 | 课前（第一步）|
| 适用级别 | HSK 1-5 |

**一句话定位**：用话题点燃好奇心——"这一课学完，你就能用中文做这件事了。"

> 这是新版本的核心创新。ClassReady 不仅是做题，更是让学生对本课内容和学习目标产生期待。

关键要素：
- 每课 1 个 R0 模块，包含 2-3 个互动问题
- 类型多样性：投票题（选你平时怎么做）、开放输入（你觉得呢）、图片刺激（看这张图想到了什么？）
- 文化 tips（HSK1-2 加英文翻译，HSK3+ 纯中文）
- **目标是让学生说一句真心话，不是答一道题**

**形式可能：**
| 形式 | 说明 |
|---|---|
| 🗳️ 快速投票 | 2 个选项，点一下就参与 |
| 💬 一句话留言 | 输入框 + 弹幕滚动上墙 |
| 🖼️ 看图说话 | 展示图片，问"你看到了什么？" |
| 🎬 短视频 | 10 秒短视频 + 一个思考问题（未来扩展）|

### R1 词义闯关 (vocab_match)

| 属性 | 值 |
|---|---|
| 代码 | R1 |
| 主练能力 | 读 |
| 状态 | live |
| 时间 | 课前（第二步）|
| 对位 ClassPro | v1 看图猜词 / v7 词了个词 |
| 适用级别 | HSK 1-5 |

**一句话定位**：全部生词配对闯关，游戏化零压力入门。

> 与旧版区别：**不再选 5-8 个核心词，而是本课全部生词都要出现**。既是预习也是第一次完整接触。

**命题原则：**
- 覆盖本课所有生词，一个不落
- 分两轮：看中文选英文 + 看英文选中文
- 做成配对游戏（翻牌配对/快速选择/拖拽连线）而非枯燥的选择题
- 加入游戏化元素：连续答对加分、进度条可视化
- **目标：让学生"玩"完一遍，生词全过了一遍脸**

**游戏形式选项：**
| 形式 | 说明 |
|---|---|
| 🃏 翻牌配对 | 翻一张中文牌，找对应的英文牌 |
| ⚡ 快速选择 | 出中文词，4 秒内选英文 |
| 🔗 拖拽连线 | 左列中文，右列英文，拖拽连线 |
| 🏆 限时闯关 | 每关 3-5 词，全对进下一关 |

### R2 跟读评分 (read_aloud)

| 属性 | 值 |
|---|---|
| 代码 | R2 |
| 主练能力 | 说 |
| 状态 | live |
| 时间 | 课前（第三步）|
| 对位 ClassPro | v2 你说我猜 / v6 描述猜词 |
| 适用级别 | HSK 1-5 |

**一句话定位**：听→读→反馈，让嘴巴先动起来。

**技术实现：**
- **不要求你自己录音**。系统支持两种音源：
  - ① **教材音频文件**（已有 audio/ 文件夹的 MP3）→ 有就用
  - ② **在线语音合成**（Web Speech API / Edge TTS）→ 没有音频时自动使用
- 学生对着手机麦克风跟读，系统评分

**命题原则：**
- HSK1-2：跟读全部生词，附带拼音
- HSK3+：跟读核心生词 + 语法句（每课 2-3 个含语法点的句子）
- 可开关拼音显示——第一次带拼音读，第二次尝试不带拼音

---

### R3 语法句感知 (grammar_sentence_read)

| 属性 | 值 |
|---|---|
| 代码 | R3 |
| 主练能力 | 读,说 |
| 状态 | live |
| 时间 | 课前 |
| 适用级别 | HSK 2-5 |

**一句话定位**：不分析语法，只读句子——"这个句子听起来是什么感觉？"

> 依据你的建议：语法不放预习分析，改为本课里跟语法相关的句子的理解和朗读

**命题原则：**
- 每课选 2-3 个含本课语法点的句子
- 每句先听/读一遍，然后答 1 道简单的理解题（检测是否懂了意思）
- **不做语法分析、不讲规则**——只是让学生"感觉一下"这个句子

---

### R4 拼音拼读 (pinyin_practice)

| 属性 | 值 |
|---|---|
| 代码 | R4 |
| 主练能力 | 字 |
| 状态 | live |
| 时间 | 课前 |
| 适用级别 | HSK 1（拼音入门阶段专属）|

**一句话定位**：HSK1 专属——从拼音规则到汉字匹配，打好语音基础。

> 根据你的要求：保留汉字↔拼音匹配，新增拼音拼读练习 + 拼音规则自学模块

包含三个子模块：
- **A. 汉字↔拼音匹配**：看汉字选拼音 / 看拼音选汉字（覆盖本课全部生词）
- **B. 拼音拼读练习**：声母 + 韵母 → 拼成完整音节（如 b + ā → bā）
- **C. 拼音规则自学**：互动式规则学习（如 jqx 遇到 ü 去点、声调标调规则等）
- 起始几课多安排 B 和 C，后续逐渐减少

---

### R5 汉字书写 (char_write)

| 属性 | 值 |
|---|---|
| 代码 | R5 |
| 主练能力 | 字,写 |
| 状态 | live |
| 时间 | 课前 |
| 适用级别 | HSK 1（带拼音）/ HSK 2（无拼音）|

**一句话定位**：手写汉字，尽快习惯"写字"这件事。

> 根据你的要求：**增加数量**——HSK1 每课写 8 个汉字，HSK2 每课写 10 个

**命题原则：**
- HSK1 每课写 8 个汉字：简单字（笔画 ≤ 8）全部要求写，复杂字只需辨认
- HSK1 提供笔顺动画 + 拼音辅助
- HSK2 去掉拼音，增至每课 10 个字
- HSK3+ 不设此题（课中 w1/w2 已覆盖）
- 输入方式：手指在屏幕上写（适合手机）或 键盘打字（适合电脑）

---

### R6 课文听力 (text_listening)

| 属性 | 值 |
|---|---|
| 代码 | R6 |
| 主练能力 | 听,读 |
| 状态 | live |
| 时间 | 课前（最后一步）|
| 对位 ClassPro | t1 挖空复述 / tt 课文判断 / t3 接龙造句 |
| 适用级别 | HSK 2-5 |

**一句话定位**：课文拆成短段，听一段答一题——熟悉内容和语音。

> 依据你的建议：还原为听力形式，拆解课文为若干短句/篇章，每段配音频+理解题

**命题原则：**
- 每课课文拆成 3-5 个短段，每段 ≤ 30 字
- 每个短段只配 1 个问题——答完就下一段
- 问题是理解型而非听写型——检测学生听懂了意思
- 先只听（不显示文本），答完题后可选择看文本对照
- 控制总时长不超过 3 分钟

## 四、完整预习流程演示（以 HSK3 第 11 课为例）

```
R0 话题预热 (1分钟)
  "你平时会随手关灯/关空调吗？" → 投票
  "学完这一课，你可以提醒朋友'别忘了关空调'" → 文化小知识

R1 词义闯关 (3分钟)
  全部 14 个生词 × 2 轮（中→英，英→中）
  翻牌配对游戏，答对有音效和进度条

R2 跟读评分 (2分钟)
  8 个核心词 + 2 个语法句
  有音频播音频，没有用 TTS

R3 语法句感知 (1分钟)
  3 个带"把"字句的句子：听→读→答 1 道理解题
  不讲规则，只感觉句子意思

R6 课文听力 (3分钟)
  4 个短段，每段听一遍 + 答 1 题

→ 总共约 10 分钟
→ 教师收到预习报告：全班"把"字句理解准确率 60% → 课上多讲
   Mwasonya "空调"选错了 → 课上多叫他回答
```
